# Köldmedium-DNA (PT-metod)

Ett indikativt verktyg för att identifiera köldmedier baserat på
tryck- och temperaturjämförelse (PT-metod).

⚠️ Ej godkänd identifieringsmetod – endast för utbildning och analys.

## Funktioner
- PT-jämförelse mot flera köldmedier
- Identifierar möjliga träffar
- Indikerar luft / icke-kondenserbara gaser

## Teknik
- Python
- Streamlit
- CoolProp
